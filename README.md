# Price Tracker

A minimal low-latency C++ TCP-based price tracking system.

## Build Instructions:
- Inside the project folder, create a build folder to store the build files
```
cd price_tracker
mkdir build
cd build
cmake ..
make
```

## Run

- run using ./pt for main project and
```
./pt
```
- and on a differnt terminal run ./sender for the tick sending tool
```
./sender
```
